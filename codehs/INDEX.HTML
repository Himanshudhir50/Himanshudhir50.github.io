CODE HS

CodeHS is an interactive online learning platform offering computer science and programming instruction for schools and individual learners. CodeHS is focused on spreading access to and knowledge of computer science by offering online instructional materials supported by remote tutors. In the introductory learning module, students on the site practice computer science concepts and programming skills by giving commands to a dog named Karel. Similar to the original Karel programming language developed by Richard E. Pattis, Karel the dog must complete various tasks by moving around a grid world, and putting down and picking up tennis balls using only simple commands. Later learning modules teach more advanced concepts using languages like JavaScript, Java, and HTML.
